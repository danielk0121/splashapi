1. 뿌리기
사용자 id : 숫자, X-USER-ID 헤더
대화방 ID : 문자, X-ROOM-ID 헤더
잔액 체크는 하지 않음

REQ : 
	- 뿌리기 금액
	- 뿌리기 인원 (대화방 인원 이하)
	- 생성 사용자 id
	- 대화방 id
RES : 
	- 뿌리기 토큰
로직 :
	- 고유한 뿌리기 토큰 생성
		- 문자열 3자리, 예측 불가 
	- 뿌릴 금액을 인원수에 맞게 분배 후 저장
		- 분배 로직 자유

--------------------
2. 받기
토큰에 해당하는 뿌리기 분배 건 중
	아직 누구에게도 할당되지 않은 분배건 하나를
	api 호출한 사용자에게 할당
	그 금액을 응답값으로 제공
뿌리기 당 한 사용자는 한번만 받을 수 있음
자신이 뿌리기한 건은 자신이 받을 수 없음
뿌리기가 호출된 대화방과 
	동일한 대화방에 속한 사용자만 받을 수 있음
뿌리기 건은 10분간만 유효함

req : 
	- 뿌리기 토큰
	- 받기 사용자 id
	- 대화방 id
res : 
	- 금액

--------------------
3. 조회
뿌리기 토큰을 요청값으로 받음
뿌리기 생성 사용자 id 만 조회 가능
	=> 에러 응답
조회 유효기간 7일
	=> 에러 응답
토큰에 해당하는 뿌리기의 현재 상태를 응답
	현재상태 : 
		- 뿌리기 생성자 id
		- 토큰
		- roomId
		- 뿌리기 금액
		- 뿌리기 생성 시간
		- 받기 완료된 금액
		- 받기 완료된 정보 리스트
			- 받기 금액
			- 받기 사용자 id
req : 
	- 뿌리기 토큰
res : 
	- 현재 상태
